This plugin will enable change author url in your Wordpress theme.

== Description ==

Authors in WordPress have a homepage url like http://domain.com/author/authorname where all their posts are shown. And many of us who think it doesnâ€™t look attractive want to change that url format. This plugin help you to change that url format.

Before: http://domain.com/author/authorname

After: http://domain.com/Your_Custom_Word/authorname

= Plugin Features =
* Easy to use.
* Only 3 kb.
* Customizable.

== Installation ==

= Installation =
1. Upload 'tcbd-author-url-changer' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
